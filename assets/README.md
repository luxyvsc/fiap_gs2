# Assets

Este diretório contém recursos visuais, imagens, prints e anexos do projeto SymbioWork.

## Estrutura Sugerida

```
assets/
├── images/           # Imagens gerais do projeto
├── screenshots/      # Screenshots das telas e funcionalidades
├── diagrams/         # Diagramas de arquitetura, fluxos, etc
├── icons/            # Ícones e logos
└── videos/           # Recursos para vídeo de demonstração (opcional)
```

## Como Usar

- Adicione screenshots das telas implementadas
- Inclua diagramas de arquitetura (use draw.io, Mermaid, etc)
- Organize por categoria
- Mantenha nomes de arquivos descritivos

## Para o PDF da GS

Inclua no PDF de entrega:
- Screenshots das principais telas
- Diagramas de arquitetura
- Gráficos de resultados (ML metrics, dashboards)
- Prints de logs/execução

## Para o Vídeo

Prepare recursos visuais para facilitar a gravação:
- Thumbnails
- Slides de introdução/conclusão (opcional)
- Screen recordings das demos
